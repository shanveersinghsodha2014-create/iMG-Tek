# iMG-Tek

**iMG-Tek** is a versatile image processing tool built with Python and Pillow.  
It supports image format conversions, downscaling, pixel manipulation, filters, metadata editing, and vector (.svg) processing.

---

## Features

- Load and save images in various formats (JPEG, PNG, BMP, TIFF, etc.)  
- Image resizing, downscaling, and pixel manipulation  
- Apply basic filters (blur, sharpen, grayscale, etc.)  
- View and edit image metadata (EXIF, etc.)  
- Basic SVG vector file support and processing  
- Command-line friendly and easy to integrate into custom apps

---

## Requirements

- Python 3.7 or higher  
- [Pillow](https://python-pillow.org/) (Python Imaging Library Fork)  
- [svgwrite](https://github.com/mozman/svgwrite) for SVG processing  
- (Optional) piexif for advanced EXIF metadata handling
